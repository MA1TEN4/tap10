##  Descripción
Este trabajo práctico tiene como objetivo introducir los fundamentos del lenguaje **JavaScript**, practicando con variables, estructuras de control, funciones, arrays, objetos y clases.

---

## Archivos y Ejercicios

| Archivo | Descripción |
|----------|--------------|
| `hola_mundo.js` | Imprime "Hola JavaScript" en consola |
| `variables.js` | Práctica con variables (`var`, `let`, `const`) y tipos de datos |
| `control_flujo.js` | Condicionales `if`, `else if` |
| `funciones.js` | Funciones para calcular área y sumar arrays |
| `estructuras_datos.js` | Trabajo con arrays y objetos |
| `clases.js` | Definición e instancia de una clase `Persona` |

---

## Instrucciones para ejecutar

Abrí una terminal y ejecutá cada archivo con **Node.js**:

```bash
node hola_mundo.js
node variables.js
node control_flujo.js
node funciones.js
node estructuras_datos.js
node clases.js
